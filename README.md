# 🎯 GamePoints

🌐 **Live Demo:** https://www.gamepoints.online/

---

## 🚀 Overview

**GamePoints** is a modern, mobile-first web application designed to track scores for group games such as board games, card games, casual competitions, and sports.

The app focuses on delivering an extremely fast, intuitive, and frictionless experience, allowing users to start a game and track scores in seconds — even in real-life social environments.

This project was built as part of a personal initiative to design, build, and ship a real product using modern web technologies and AI-assisted development workflows.

---

## ✨ Features

- ⚡ **Instant Game Creation** – Start tracking scores in seconds
- 👥 **Multiple Players** – Add and manage players easily
- ➕➖ **Quick Score Updates** – Fast increment/decrement controls
- 🔄 **Undo Functionality** – Revert the last action instantly
- 📊 **Action History** – Track score changes in real time
- 🏆 **Live Ranking** – Players are dynamically sorted by score
- 📱 **Mobile-First UX** – Optimized for one-hand usage in real scenarios
- 🌍 **Internationalization Ready** – Auto language detection + manual selection

---

## 🧱 Tech Stack

- **Frontend:** React / Next.js  
- **Styling:** Tailwind CSS  
- **Deployment:** Vercel  
- **Version Control:** GitHub  

### 🧠 AI-Assisted Development

- Cursor (AI-powered IDE)
- ChatGPT, Claude, Gemini
- Lovable (used for initial MVP generation)

---

## 🧪 Development Approach

This project follows a **rapid MVP → iterate → refine** methodology:

1. **Initial MVP built with Lovable** to accelerate development
2. **UX/UI improvements** based on real-world usage scenarios
3. **Refactoring and enhancements** using AI-assisted tools
4. **Continuous deployment** via Vercel connected to GitHub
